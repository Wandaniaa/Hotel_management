Hotel Management System

A Django-based Hotel Management System designed to help hotels manage daily operations digitally, 
including room and hall management, guest check-in/check-out, payment processing, invoice generation, and transaction reports.

-- Main Features --
# Room Management
- Add, edit, and delete room data
- Automatic room status updates:
  Available
  Occupied
  Cleaning
- Room type and pricing management

# Hall Management
- Hall data management
- Hall reservation system
- Hall usage status monitoring

# Guest Management
- Store guest information
- Guest check-in and check-out history
- Guest notes and additional information
  
# Check-In System
- Guest registration form
- Room or hall selection
- Automatic total cost calculation
  Supports:
  - Percentage discounts
  - Fixed amount discounts
  - Automatic invoice generation
  
# Check-Out System
- Payment calculation
- Automatic room status updates
- Supports overpayment with automatic change calculation
- Transaction history storage
  
# Cleaning Management
- Room and hall cleaning logs
- Cleaning activity tracking
- Cleaning status monitoring

# Invoice & Reporting
- Payment invoice generation
- Transaction reports
- Check-in and check-out reports
- Hotel revenue reports
